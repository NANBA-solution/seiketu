# App Store 審査用メモ（セイケツ）

## アプリ概要

身だしなみケアのリマインドアプリ。アカウント登録なし。データは端末内のみ保存。

## 審査用アカウント

不要（ログイン機能なし）

## 通知の確認手順

1. アプリ起動 → オンボーディング完了 → 通知を許可
2. **設定**タブ → カテゴリーの通知を **ON**
3. Debug ビルドのみ: 「5秒後にテスト通知」で動作確認
4. Release ビルド: 期限日を今日に手動設定し、9:00 / 19:00 のローカル通知を確認

## プライバシー（App Store Connect 入力目安）

- **データ収集**: なし（第三者共有・トラッキングなし）
- **UserDefaults**: ケア記録・設定の端末内保存のみ（Privacy Manifest 同梱）
- **プライバシーポリシー URL**: App Store Connect に掲載する URL と、アプリ内「プライバシーポリシー」の内容を一致させること

## 暗号化

`ITSAppUsesNonExemptEncryption = NO`（標準 HTTPS のみ、独自暗号なし）

## サポート連絡先

nanbacoltd.95@gmail.com（アプリ内・利用規約に記載）

## データ削除

設定 → 「データをすべて削除」で端末内データを消去可能
