# セイケツ (SEIKETSU)

ほっとくと、おっさんになる。だらしない男のための全自動・身だしなみアラートアプリ。

## 概要

インストールして通知を許可するだけで、爪・鼻毛・眉・髭・耳毛などの手入れ周期が自動でセットされ、最適なタイミングでアラートが届きます。「やった」を押すだけで周期を学習し、通知タイミングを最適化します。

## 技術スタック

- **フロントエンド**: Swift / SwiftUI（iOS 先行）
- **通知**: iOS ローカル通知（サーバー不要）
- **データ**: 端末内ローカル保存
